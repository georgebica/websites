# Generated Sites

66 business sites — deploy as a single Cloudflare Pages project.

## Deploy on Cloudflare Pages

1. Create a new Cloudflare Pages project
2. Connect this GitHub repo
3. Set **Framework preset**: None
4. Leave **Build command** empty
5. Set **Build output directory**: `.` (root)

Each site is accessible at `your-domain.com/business-name/`.
The root page (`/`) lists all sites with search.
