// Operațiuni

Ce trecem pe factură.

// 01

Diagnoză OBD

Citire erori, motor, cutie, ABS, airbag. Cu tester profesional.

// 02

Mecanică generală

Schimburi de ulei, distribuție, ambreiaj, rulmenți, transmisie.

// 03

Electrică auto

Cablaje, conectori, ECU, baterii, demaroare, alternatoare.

// 04

Sistem frânare

Plăcute, discuri, etrieri, lichid de frână - schimb și ajustare.

// 05

Suspensii

Amortizoare, articulații, geometria roților, capete bară direcție.

// 06

Climatizare

Încărcare freon, verificare circuit AC, reparații compresor.
